## `git reset` && `git push` with force overwrite

```sh
# master commit uid
$ git reset --hard b9e3f7f2f66674901b467183dd81d493e50677f5

```

```sh
# git push branch_name --force

# local master
$ git push origin master --force

# remote master
$ git push origin/master -f

```
